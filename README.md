# Система проверки технической документации (Федеральное казначейство)

Данное приложение предназначено для автоматизированной проверки технической документации (в форматах DOCX, TXT) на соответствие заданным требованиям. Оно позволяет анализировать документы на предмет импортозамещения, наличия/отсутствия конкретных фрагментов текста, сравнения версий программного обеспечения и выполнения сложных комбинированных условий.

Проект разработан для нужд Федерального казначейства и помогает специалистам быстро выявлять несоответствия в документации, экономя время и снижая риск ошибок.

## Возможности

-   **Загрузка документов:** Поддержка файлов `.docx` и `.txt`.
-   **Гибкая система проверок:** Создание, редактирование и удаление проверок через удобный графический интерфейс.
-   **Разнообразные типы проверок:**
    -   `no_text_present`: Текст **не должен** содержать заданные слова/фразы.
    -   `text_present`: Текст **должен** содержать заданные слова/фразы.
    -   `text_present_without`: Текст должен содержать одни слова, но **не содержать** другие.
    -   `fuzzy_text_present`: Нечеткий поиск фразы (с учетом опечаток).
    -   `no_fuzzy_text_present`: Нечеткая проверка отсутствия фразы.
    -   `text_present_in_any_table`: Поиск фразы внутри таблиц документа.
    -   `version_comparison`: Сравнение найденных версий ПО/оборудования с требуемыми с использованием операторов (`>`, `>=`, `=`, `<`, `<=`, `!=`).
    -   `combined_check`: Объединение нескольких условий в одно с логикой `И`/`ИЛИ`.
-   **Управление проверками:** Возможность группировать, включать/отключать, дублировать и импортировать/экспортировать проверки.
-   **Детальные результаты:** Подробный отчет о каждой проверке с указанием статуса, страницы, контекста и процента совпадения для нечеткого поиска.
-   **Просмотр документа с подсветкой:** Встроенный просмотрщик с цветовой подсветкой (желтым - строка, красным - проблемное слово) и подсказками по исправлению.
-   **Анализ версий ПО:** Специализированный диалог для поиска и сравнения версий программного обеспечения, упомянутых в документе, с данными из Excel-файлов (реестров ПО).
-   **Настройка интерфейса:** Выбор темы оформления (Темная, Светлая, Смешанная).
-   **Сохранение настроек:** Все настройки, включая тему и пороги нечеткого поиска, сохраняются между сеансами.

## Технологический стек

-   **Язык:** Python 3.8+
-   **GUI:** PyQt6
-   **Обработка текста:** Regular Expressions (`re`), `rapidfuzz` (для нечеткого поиска)
-   **Работа с конфигурацией:** `PyYAML`
-   **Парсинг DOCX:** Встроенные средства работы с zip-архивами и XML (`zipfile`, `xml.etree.ElementTree`)
-   **Работа с Excel:** `pandas`, `openpyxl`
